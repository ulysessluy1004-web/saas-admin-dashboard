# SaaS Admin Dashboard

A modern SaaS Admin Dashboard portfolio project built with Next.js App Router, TypeScript, and Tailwind CSS. The project focuses on clean UI design, responsive layouts, reusable dashboard structure, and realistic mock data.

This is a frontend-only project. It does not include a backend, database, or real authentication yet.

## Live Demo

Live demo: [https://saas-admin-dashboard-eta.vercel.app/](https://saas-admin-dashboard-eta.vercel.app/)

Repository: [https://github.com/SeniorDeveloperFullStack/saas-admin-dashboard](https://github.com/SeniorDeveloperFullStack/saas-admin-dashboard)

## Features

- Responsive SaaS dashboard layout
- Sidebar navigation and top navbar
- Overview dashboard with stats, revenue chart placeholder, recent customers, and activity feed
- Customers page with customer stats and customer table
- Analytics page with metric cards and chart placeholders
- Billing page with plan cards, invoices table, and payment status
- Settings page with profile, workspace, and notification settings UI
- Login and signup UI pages
- Interactive mock notification dropdown
- Modern animation effects and hover states
- Mock data only, ready to connect to real APIs later

## How To Run Locally

1. Clone the repository:

```bash
git clone https://github.com/SeniorDeveloperFullStack/saas-admin-dashboard.git
```

2. Go into the project folder:

```bash
cd saas-admin-dashboard
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open the app in your browser:

```text
http://localhost:3000
```

## Build For Production

Run:

```bash
npm run build
```

Then start the production server:

```bash
npm run start
```
